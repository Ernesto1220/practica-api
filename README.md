## Requisitos

- Python 3.x
- Biblioteca 'requests'

## Instalación

1. Clona este repositorio.
2. Instlaa las dependencias con 'pip install requests'.
3. Configura tu clave de API en 'config.py'.

## Uso

Ejecuta 'main.py' para iniciar la calculadora de divisas.
